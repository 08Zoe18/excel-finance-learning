# Excel 财务学习

本仓库用于记录Excel学习过程，目标：掌握财务常用函数、透视表、Power Query。

## Day 1（2026.03.05）
- 练习内容：SUM、AVERAGE、IF函数
- 文件：Day1_收支表.xlsx
## Day 2（2026.03.06）
- 练习内容：从金蝶导出3月明细账，用 Excel 做产品毛利分析（收入、成本、毛利、毛利率），并制作柱状图
- 文件：[3月产品分析_练习.xlsx](./3月产品分析_练习.xlsx)
- 发现：C产品毛利率30%，低于A/B的40%
## Day 3（2026.03.07）
- 学习内容：数据透视表基础
- 练习文件：[Excel_Day3_透视表练习.xlsx](./Day3/Excel_Day3_透视表练习.xlsx)
- 收获：用透视表快速汇总3月各产品收入、成本及费用，验证了手动分析结果。
## Day 4（2026.03.08）
- 学习内容：VLOOKUP 进阶、INDEX+MATCH、辅助列、数据清洗函数（ABS, IF, EXACT, TRIM, TEXT）
- 练习文件：[Day4VLOOKUP练习.xlsx](./Day4/银行对账练习.xlsx)
- 函数总结：[Excel_Functions_Summary.md](./Day4/Excel_Functions_Summary.md)
## Day 5（2026.03.09）
- **学习内容**：Power Query 基础——从文件夹合并多个 Excel 文件，清洗数据。
- **练习文件**：[PowerQuery_练习.xlsx](./Day5/PowerQuery_练习.xlsx)
- **核心收获**：
  - 学会用 Power Query 连接文件夹，自动合并所有文件的数据。
  - 掌握合并后修改列名、调整数据类型、删除不需要的列等清洗操作。
  - 理解“刷新”功能：以后文件夹里新增文件，只需在 Excel 中右键刷新，数据自动更新。

## Day 6（2026.03.10）
- **学习内容**：Power Query 进阶（合并查询、追加查询、分组依据、透视列、逆透视列、自定义列）
- **练习文件**：[PowerQuery_进阶.xlsx](./Day6/PowerQuery_进阶.xlsx)
- **核心收获**：
  - 掌握用 Power Query 实现类似 SQL JOIN 的多表合并
  - 能够纵向追加多个相同结构的表
  - 学会分组汇总和透视列，快速生成交叉分析表
  - 理解逆透视列在数据整理中的作用
  - 能够编写简单的 M 公式进行自定义计算
  - 操作步骤
  - 合并查询：主页 → 合并查询 → 选择匹配列和连接类型
  - 追加查询：主页 → 追加查询 → 选择要合并的表
  - 分组依据：主页 → 分组依据 → 设置分组列和聚合
  - 透视列：转换 → 透视列 → 选择值列和聚合函数
  - 自定义列：添加列 → 自定义列 → 输入 M 公式
## Day 7（2026.03.11）
- **学习内容**：Power Query 进阶（合并查询 + 透视列）
- **练习文件**：[PowerQuery_进阶练习.xlsx](./Day7/PowerQuery_进阶练习.xlsx)
- **核心收获**：
  - 掌握用合并查询将两个表通过关键字段关联（类似 VLOOKUP）
  - 学会用自定义列计算新字段（销售额）
  - 能够用透视列生成区域销售交叉表
  - 理解 Power Query 的步骤记录机制，数据更新后只需刷新即可重算
- **操作步骤概览**：
  1. 从工作簿加载两个表到 Power Query
  2. 合并查询（左外部）匹配产品信息
  3. 添加自定义列计算销售额
  4. 透视列生成区域销售额矩阵
  5. 关闭并上载到 Excel
